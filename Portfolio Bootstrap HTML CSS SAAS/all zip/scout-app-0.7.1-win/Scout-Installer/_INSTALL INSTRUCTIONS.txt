1. Run "Air-Java7.exe"
2. Run "Scout.exe" once and then close it.
3. Run the following command, replace PATH_TO_SCOUT with the actual folder:

msiexec /i "PATH_TO_SCOUT\setup.msi" TARGETDIR="C:\Program Files\Scout" /qb

You can now delete the "Scout-Installer" folder.

1. Run "Air-Java7.exe"
